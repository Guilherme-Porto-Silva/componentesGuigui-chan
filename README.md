# **🎨 componentesGuigui-chan**

Um projeto desenvolvido para a disciplina de **Programação Orientada a Objetos II** na **Universidade Estadual de Minas Gerais (UEMG)**. O objetivo deste repositório é centralizar a criação e o estudo de componentes de interface gráfica (GUI) utilizando a linguagem Java.

## **📌 Sobre o Projeto**

O componentesGuigui-chan surgiu como uma atividade acadêmica para explorar os conceitos de POO aplicados ao desenvolvimento de interfaces. Aqui, focamos em:

* Encapsulamento de lógica de componentes.  
* Herança e polimorfismo em elementos de interface.  
* Tratamento de eventos (ActionListeners, MouseEvents, etc.).  
* Organização de layouts.

## **🚀 Tecnologias Utilizadas**

* **Linguagem:** Java (JDK 17+ recomendado)  
* **Biblioteca de GUI:** Swing / AWT  
* **IDE recomendada:** IntelliJ IDEA / Eclipse / VS Code
